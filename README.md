# MD-Office-Assistant 办公助手

## 说明

MD-Office-Assistant致力于打造个性化的文件处理工具

## 特性

- [x] Excel智能合并
- [x] Email自动附件下载

## 安装和使用
pip install -r requirements.txt
 
python main.py
